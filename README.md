# vue-guides
vue-guides is a lightweight, simple and customizable tour plugin for use with Vue.js.

## Getting Started
You can install vue-guides using npm or by downloading the minified build on GitHub.

```
npm install vue-guides
```

## Features
1. [ ] 动画平滑跳转到待指引区域；
2. [ ] 高亮展示该区域
3. [ ] 可自定义指引图标和描述文字
4. [ ] 有导航按钮（上一步、下一步）
5. [ ] 或可支持路由跳转指引
6. [ ] 支持页面[data-*]属性
